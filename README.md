# Jay Chaudhari — Portfolio

Premium cinematic developer portfolio built with React + TypeScript + Framer Motion + Tailwind CSS.

## Features
- Custom animated cursor with spring physics
- Particle network background
- Mouse-following spotlight effect
- Cinematic loading screen
- Scroll progress indicator
- Typewriter role switcher
- Animated counters
- Timeline journey section
- Interactive 3D skill cards with progress bars
- AgroAI featured project showcase
- Copy-to-clipboard email button
- Contact form with success animation
- Fully responsive

## Tech Stack
- React 19 + TypeScript + Vite
- Framer Motion (all animations)
- Tailwind CSS v3
- Lucide React (icons)

## Run locally
```bash
npm install
npm run dev
```

## Deploy to Vercel
1. Push to GitHub
2. Import repo in vercel.com
3. Deploy — zero config needed

## Deploy to GitHub Pages
```bash
npm run build
# Push the /dist folder
```

## Customize
- Edit `src/App.tsx` — all content (name, skills, projects, timeline, contact) is in the DATA section at top
- Edit `src/index.css` — all colors, fonts, animations
